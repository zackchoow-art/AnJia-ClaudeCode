# Aether Admin Portal v2.0

Real Estate Development Platform (REDP) - Management Portal

## Features

- **Dashboard**: Real-time KPIs and recent activities
- **Projects**: Manage development projects with full lifecycle tracking
- **Customers**: CRM for property buyers and investors
- **Sales**: Marketing campaigns and property inventory management
- **Contracts**: Contract repository with signature tracking
- **Approvals**: Payment approval workflow with checklist
- **Budget**: Cost budgeting and expense tracking
- **Finance**: Financial statements and payment schedules
- **Settings**: System configuration and audit trail

## Tech Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS 4.1
- Lucide React Icons
- React Router DOM

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The app will be available at `http://localhost:3001`
