# Phase 2 任务概述

**目标**: 将 REDP 从后台管理系统扩展为完整的房地产销售运营平台，增加房源管理、CRM、销售AI和移动端  
**启动前提**: Phase 1 验收通过  
**计划周期**: 2026-11 至 2027-03（约5个月）  
**核心价值**: 销售团队从"被系统监控"变为"被系统赋能"

---

## Phase 2 技术约束

1. **API 优先**: 所有功能先以 Edge Function API 实现，移动端是消费方
2. **维语编码**: 使用 UTF-8，数据库字段支持多语言（JSONB 存储 `{zh: "...", ug: "..."}` 格式）
3. **离线支持**: 移动端关键功能需支持弱网络环境（喀什地区网络条件）
4. **数据向前兼容**: Phase 2 的变更不能破坏 Phase 0/1 的任何接口
